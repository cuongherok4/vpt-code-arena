# Module 1: Learn (Học)

Mô hình tham chiếu: W3Schools (https://www.w3schools.com/java/default.asp)

## Nguyên tắc cốt lõi
- **Không chấm theo tốc độ.** Mục tiêu là học và hiểu, không thi đấu.
- Tiến trình học là **tuyến tính và có khóa**: phải hoàn thành chương trước mới mở chương sau.

## Cấu trúc nội dung
- Menu dạng cây bên trái, chia nhóm lớn: Basics, OOP, Collections, Advanced...
- Mỗi nhóm gồm nhiều bài học nhỏ theo thứ tự tuyến tính.

## Trang bài học (Lesson Page)
- Nội dung lý thuyết.
- Mini code editor **"Try it Yourself"**: chạy thử code ngay trong trang, hỗ trợ nhiều ngôn ngữ (Java, Python, C++...).
- Nút [Chạy] [Reset] [Xóa].

## Code Challenge (cuối chương)
- Bài kiểm tra nhỏ để xác nhận đã hiểu bài trước khi mở khóa chương tiếp theo.
- Auto-grade bằng test case, feedback Correct/Wrong.

## Theo dõi tiến độ
- Đánh dấu bài đã hoàn thành (✓/✗).
- % hoàn thành lộ trình tổng thể.
- Badge/Achievement khi hoàn thành chương.
- Ghi nhận thời gian học (ngày, giờ) phục vụ thống kê cá nhân.

## Phạm vi API/Backend cần có
- CRUD lessons/chapters (thường do Admin quản lý).
- Lưu progress theo user (bài nào đã hoàn thành, chương nào đã mở khóa).
- Trình chạy code đơn giản (mini code runner) cho "Try it Yourself" — tách biệt với Judge0 dùng cho Exam/Battle vì không cần chấm test case phức tạp, chỉ cần show output.
- Auto-grade Code Challenge (test case đơn giản, không cần đo thời gian chạy).
