# Hướng dẫn đọc tài liệu dự án

Thứ tự đọc khuyến nghị cho người mới (hoặc bot) tham gia dự án:

1. `/CLAUDE.md` — tổng quan dự án, bức tranh toàn cảnh, doc map.
2. `/PROJECT_REQUIREMENTS.md` — danh sách đầy đủ chức năng + độ ưu tiên.
3. `/docs/system-design/tech-stack.md` + `database-schema.md` — nền tảng kỹ thuật.
4. `/docs/features/*.md` — chi tiết từng module (Learn, Exam, Battle, Messaging).
5. `/docs/deployment/roadmap.md` — lộ trình, tiến độ, rủi ro.

Không cần đọc lại các bản nháp/checklist cũ — mọi nội dung đã được gộp và cập nhật vào các file trên.
